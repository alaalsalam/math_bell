# Product Map — World Experience

## Target Product Shape
- Student app behaves like one lightweight game world, not a multi-dashboard system.
- Route complexity is hidden from kids; interaction feels like one continuous journey.

## Game World Structure

### 1) Regions
- `غابة الجمع` (Addition)
- `بحر الطرح` (Subtraction)
- `قلعة العمودي` (Vertical)
- `جزيرة الكسور` (Fractions)

### 2) Skills
- كل منطقة فيها مهارات مرتبة من السهل إلى الأصعب.
- كل مهارة تعرض حالة واحدة واضحة:
  - `ابدأ`
  - `تابع`
  - `مكتمل 👑`

### 3) Sessions
- نوعان فقط للطالب:
  - `تدريب سريع` (افتراضي)
  - `حصة الجرس`
- التحدي السريع (`5 أسئلة`) يظهر كخيار ثانوي داخل المهارة نفسها.

### 4) Rewards
- مكافأة فورية بعد كل إجابة صحيحة.
- مكافأة نهاية الجلسة: نجوم + سلسلة + رسالة من الأستاذة عائشه شفلوت الحارثي.
- مكافآت أسبوعية خفيفة (دون جداول معقدة).

## Journey Maps

## Student Journey — First-Time User
1. فتح التطبيق → شاشة ترحيب بعلامة `إعداد وتقديم: الأستاذة عائشه شفلوت الحارثي`.
2. دخول سريع (اسم + كلمة مرور بسيطة).
3. دخول عالم اللعب مباشرة.
4. اختيار منطقة.
5. اختيار مهارة واحدة مقترحة (CTA واضح).
6. بدء أول جلسة.
7. شاشة مكافأة مختصرة + زر "العب مرة ثانية".

### Success Signal
- أول سؤال يبدأ خلال أقل من 60 ثانية من أول دخول.

## Student Journey — Daily User
1. فتح التطبيق.
2. زر كبير: `كمّل من آخر مهارة`.
3. لعب جلسة قصيرة.
4. مكافأة + تقدم السلسلة اليومية.

### Success Signal
- 2–3 دقائق لعب يومي بدون ضياع في التنقل.

## Student Journey — Struggling User
1. النظام يوجّه الطالب تلقائيًا لمهارة دعم.
2. تفعيل تلميحات أبسط + خطوات مرئية.
3. جولة أسئلة أقصر.
4. مكافأة تشجيعية حتى مع تحسن جزئي.

### Success Signal
- انخفاض الخروج المبكر من الجلسة.

## Teacher Journey — Teacher Mode (Light)
1. دخول Teacher Mode من داخل الواجهة (بدون Desk).
2. تعديل إعدادات خفيفة محلية:
   - الصوت
   - المؤثرات
   - مدة الجرس
   - طول الجولة
3. عرض ملخص بسيط:
   - من يحتاج دعم الآن
   - آخر نشاط
4. رجوع سريع لوضع الطالب.

### Success Signal
- المعلمة تضبط الإعدادات خلال أقل من 30 ثانية.

## Student Screens (Max 10)
1. **Splash Aisha**
   - هوية المنتج + دخول سريع.
2. **Login / Join**
   - حقول بسيطة جدًا.
3. **Welcome Back**
   - زر `ابدأ` أو `كمّل`.
4. **World Map**
   - 4 مناطق فقط.
5. **Skill Sheet**
   - بطاقة مهارة + اختيار نوع اللعب.
6. **Play Session**
   - السؤال الحالي + تقدم + تلميح.
7. **Session Reward**
   - نجوم، سلسلة، CTA متابعة.
8. **Daily Streak Card**
   - ظهور خفيف بعد أول جلسة يومية.
9. **Mini Progress**
   - نظرة مختصرة جدًا على الإنجاز (بدون تحليلات كثيفة).

## De-Emphasis in Student Experience
- إخفاء أو تقليل المسارات القديمة المتعددة (`/g/:grade`, `/g/:grade/d/:domain`, dashboard الكثيف) من رحلة الطفل الأساسية.
- أي شاشة تحليلية تفصيلية تبقى داخل Teacher Mode فقط.
